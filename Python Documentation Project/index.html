Introduction

Python is a high-level, interpreted, and general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant indentation.

  • It is dynamically-typed and garbage-collected.
  • It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.
Python Basics

Before diving deeper, let's explore some basics:

  • Python uses indentation to define blocks of code, unlike other languages that use braces or keywords.
  • Statements end with a newline, not a semicolon.
  • Python files have a ".py" extension.

Example of a simple Python program:

print("Hello, World!")
Data Types

Python supports several standard data types:

  • int – Integers, like 42 or -13
  • float – Floating-point numbers, like 3.14
  • str – Strings, like "Hello World"
  • bool – Boolean values, either True or False
  • list – Ordered collections of items
  • dict – Key-value pairs (dictionaries)

Here is an example of defining variables in Python:

age = 25
name = "John"
is_student = True
Control Structures

Python includes common control structures like conditional statements and loops:

  • If-Else
  • For Loop
  • While Loop

Example of an if-else statement:

if age >= 18:
    print("Adult")
else:
    print("Minor")
Functions

Functions allow code to be modular and reusable. A simple function looks like this:

def greet(name):
    print(f"Hello, {name}!")

You can call this function using:

greet("Alice")
Modules and Packages

Python has a rich ecosystem of libraries and modules. A module is a file containing Python definitions and statements. You can import modules using the import keyword.

import math

Example of using a module:

import math
print(math.sqrt(16)) # Outputs: 4.0
Object Oriented Programming

Python supports object-oriented programming (OOP). You can create classes and instantiate objects:

class Person:
    def __init__(self, name):
        self.name = name
    def greet(self):
        print(f"Hello, {self.name}!")

Example of creating an object:

p = Person("John")
p.greet() # Outputs: Hello, John!
Exception Handling

Python uses the try and except blocks to handle exceptions. Here's an example:

try:
    x = 1 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
